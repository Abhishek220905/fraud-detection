# 🚀 Fraud Detection System

An AI-powered web application to detect fraudulent loan applications using real-time data analysis.

---

## 🔗 Live Demo
Frontend: https://fraud-detection-1-31eh.onrender.com  
Backend: https://fraud-detection-sfak.onrender.com

---

## ⚙️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Deployment: Render

---

## 📌 Features
- Real-time fraud detection
- Aadhaar validation
- Risk scoring system
- Live activity tracking
- Data visualization
